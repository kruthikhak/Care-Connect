K-Nearest Neighbors (KNN) Algorithm

Finding the 5 nearest hospitals to a user's location

How it works:

  1. Get user's current location (red dot)
  2. Calculate distances to all hospitals (blue dots)
  3. Sort hospitals by distance
  4. Select k nearest hospitals (highlighted)

const findKNearest = (userLocation, hospitals, k = 5) => {
    return hospitals
        .map(hospital => ({
            ...hospital,
            distance: calculateDistance(userLocation, hospital)
        }))
        .sort((a, b) => a.distance - b.distance)
        .slice(0, k);
};

Content-Based Filtering (CBF)

Recommending doctors based on user preferences and specialties

User Preferences
  • Specialty: Cardiology
  • Min Rating: 4.0
  • Location: San Francisco

How it works:

  1. Extract user preferences
  2. Calculate similarity scores for each doctor
  3. Sort by similarity score
  4. Return top recommendations

const calculateSimilarity = (preferences, doctor) => {
    let score = 0;
    if (doctor.specialty === preferences.specialty) score += 3;
    if (doctor.rating >= preferences.minRating) score += 2;
    if (doctor.city === preferences.city) score += 2;
    return score;
};
